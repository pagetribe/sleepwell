# **App Name**: SleepWell

## Core Features:

- Time Logging: Log bedtime and wake-up time using an intuitive time selector.
- Mood Rating: Rate daily mood using a simple scale (emojis or stars) both at bedtime and wake-up.
- Dream Logging: Option to record whether you woke up during a dream.
- Fuzziness Scale: Use a scale (slider) to indicate mental fuzziness level upon waking.
- Sleep Duration Calculation: Automatically calculate and display the sleep duration based on logged times.
- Local Storage: Store sleep and mood data locally, ensuring user privacy and offline access.
- Sentiment Tracker: Use an AI tool that scans sleep entries, looking for repeated words or phrases which seem related to your mood.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke calmness and rest.
- Background color: Light gray (#F5F5F5) for a clean, non-intrusive backdrop.
- Accent color: Muted teal (#80CBC4) to add a touch of freshness and highlight important actions.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Use simple, outlined icons to represent moods, sleep quality, and actions within the app.
- Employ a clean and spacious layout with generous padding to reduce visual clutter and improve usability.
